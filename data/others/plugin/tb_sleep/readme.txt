https://plugin.tyrano.jp?en=1

■ How to Use
Requires TyranoBuilder v182 or later.

From the top menu, open Project > Plugins
Click Add New and select the downloaded .tbp file.

The plugin should now be available for use.

■ Warning
Only use plugins obtained from trusted sources. 
Malicious plug-in may cause damage to or misuse of your PC.
We strongly encourage using only plugins posted on the official TyranoBuilder official plugins website.

ティラノビルダー公式プラグイン
https://plugin.tyrano.jp

■使い方
ティラノビルダーV182以上が必要です。
外部プラグイン機能を使用してください。

トップメニューから
[プロジェクト] - [外部プラグイン] を開いて
「外部プラグインを追加」をクリック。ダウンロードしたtbpファイルを選択してください。

成功すれば、プラグインを使用することが出来るようになります。

■警告
信頼できるサイトから入手したプラグイン以外は
絶対に使用しないでください。

悪意のあるプラグインによって、PCが攻撃される可能性があります。

公式サイトに乗っているプラグインの使用を強く勧めます。
