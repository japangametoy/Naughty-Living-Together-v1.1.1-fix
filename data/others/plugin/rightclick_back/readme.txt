【右クリックとかで戻るプラグイン】

■できること
・右クリックで指定したシナリオ・ラベルへジャンプ
・任意のクリックまたはキーイベントで指定したシナリオ・ラベルへジャンプ
・任意のクリックまたはキーイベントで指定したJS式を実行


■使い方
このテキストが入っているフォルダごと「data/others/plugin」フォルダに置きます。
それからfirst.ksとかに以下のように記述してください。

[plugin name=rightclick_back]
指定可能属性：なし

記述した時点から後述するタグが使用できるようになります。


■使用可能タグ
back_set        ：右クリックで指定したシナリオ・ラベルへジャンプ
使用可能属性
storage     ：ジャンプ先のシナリオを指定
target      ：ジャンプ先のラベルを指定

back_delete     ：右クリックイベントを消去
使用可能属性
なし

event_set       ：任意のクリック・キーイベントをセット
使用可能属性
name（必須）：クリックイベントの名前を指定（英数字のみ）
type        ："mouse" または "key" を指定
mouse       ：マウスのボタン番号を指定。1：左、2：中央、3：右のいずれか
key         ：キー名を指定。キー名についてはkeyConfig.tjsを参照
storage     ：ジャンプ先のシナリオを指定
target      ：ジャンプ先のラベルを指定
exp         ：イベント発生時に実行するJS式を指定
once        ：イベントを一度だけ発生させる場合はtrueを指定。初期値はtrue
default     ：対象のキーのティラノ側の既定イベントを実行しない場合はfalseを指定。初期値はtrue

event_delete    ：任意のクリック・キーイベントを消去
使用可能属性
name（必須）：event_setタグで指定したnameを指定
