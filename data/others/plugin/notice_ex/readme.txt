【通知ポップアップ表示プラグイン】


■できること
・ゲーム画面上に通知ポップアップを表示します。


■使い方
このテキストが入っているフォルダごと「data/others/plugin」フォルダに置きます。
それからfirst.ksとかに以下のように記述してください。
記述した時点から後述のタグが使用可能になります。

[plugin name=notice_ex]
指定可能属性
なし


■使用可能タグ
notice  ：通知表示
属性
text            ：通知として表示する文字列を指定します。HTMLを記述可能です。
name            ：通知ポップアップに名前をつけます。
color           ：通知の色を指定します。初期値はblackです。black, gray, white, orange, red, blue, rosy, green, pinkが指定できます。
top             ：画面上端からの位置を指定します。（単位：ピクセル）
left            ：画面左端からの位置を指定します。（単位：ピクセル）
bottom          ：画面下端からの位置を指定します。（単位：ピクセル）
right           ：画面右端からの位置を指定します。（単位：ピクセル）
width           ：通知ポップアップの幅を指定します。（単位：ピクセル）初期値は400です。
height          ：通知ポップアップの高さを指定します。（単位：ピクセル）初期値は60です。
font_color      ：文字色を指定します。
size            ：文字サイズを指定します。初期値は30です。
face            ：フォント種類を指定します。
align           ：文字揃え方向を指定します。初期値はleftです。left, center, rightが指定できます。
graphic         ：通知に表示する画像パスを指定します。画像はimageフォルダに配置します。
graphic_align   ：画像を配置する位置を指定します。top, bottom, left, rightが指定できます。
border_width    ：ボーターの幅を指定します。初期値は0です。
border_color    ：ボーター色を指定します。初期値はblackです。
radius          ：通知ポップアップの角丸を指定します。初期値は0です。
bgcolor         ：通知ポップアップの背景色を指定します。bgcolorが指定されている場合、colorパラメータの値は無視されます。
method          ：通知ポップアップの表示／非表示アニメーションを指定します。bgタグのmethodと同じものが指定できます。初期値はfadeInです。
fade_time       ：通知ポップアップの表示／非表示フェード時間を指定します。（単位：ミリ秒）初期値は200です。
wait_time       ：通知ポップアップが表示されてから非表示になるまでの待機時間を指定します。（単位：ミリ秒）初期値は1000です。
wait            ：通知ポップアップが非表示になるまで処理を待つ場合はtrue、待たずに処理を進める場合はfalseを指定します。初期値はtrueです。


■使い方のコツ
topとbottom両方に「0」を指定すると、上下方向の中央にポップアップが表示されます。
leftとright両方に「0」を指定すると、左右方向の中央にポップアップが表示されます。
top, left, bottom, rightすべてに「0」を指定すると、画面中央にポップアップが表示されます。
